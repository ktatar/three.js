[name]

AudioGenerator loops an audio buffer to create an audio oscillator, similar to audio synths in early games.

This uses the [link:https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API Web Audio API].

Code Example

// Create an AudioListener const listener = new THREE.AudioListener(); // Create an AudioInstrument const myInstrument = new AudioInstrument( listener ); // Create an AudioBuffer const myBuffer = new AudioBuffer( listener ); myBuffer.createBuffer( 1 , 1 ); myBuffer.sawtooth( 1 ).add(); // Create an AudioGenerator, similar to an audio oscillator const myGenerator = new AudioGenerator( listener ); myGenerator.connect( listener.getInput() ); // Set the frequency and the gain of the AudioGenerator myGenerator.playbackRate = 432; myGenerator.loop = true; myGenerator.setGain( 0.05 ); // Make a sound using the AudioGenerator and the AudioBuffer myGenerator.start( 0 + myGenerator.context.currentTime , myBuffer.buffer ); myGenerator.stop( 0.5 + myGenerator.context.currentTime );
